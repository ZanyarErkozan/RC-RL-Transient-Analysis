# RC and RL Transient Analysis

**Author:** Zanyar Erkozan

---

## 📋 Overview
This advanced project focuses on the dynamic behavior of **RC and RL Circuits**, specifically analyzing their transient responses and time constants.

## ⚙️ Core Concepts Covered
- **Transient Analysis:** Step-response of capacitors and inductors in direct current (DC) circuits.
- **Time Constants (Tau):** Theoretical calculation, oscilloscope measurement, and error analysis.
- **First-Order Circuits:** Natural and forced responses over time.
- **Oscilloscope Operations:** Capturing and analyzing rapid voltage/current changes.

## 📁 Files Included
- `RC-RL-Transient-Analysis.pdf`: The finalized report featuring measured discharge/charge curves, error percentage calculations, and theoretical alignment.
